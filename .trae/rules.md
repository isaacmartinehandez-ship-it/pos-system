# Reglas del Proyecto POS System

## Descripción General
Este es un sistema de Punto de Venta (POS) inspirado en SimplyGest, construido con Next.js 15, TypeScript y Tailwind CSS.

## Estructura del Proyecto
- `src/app/`: Páginas y rutas de Next.js (App Router)
- `src/components/`: Componentes reutilizables
- `src/store/`: Gestión de estado global con Zustand
- `.trae/`: Configuraciones y reglas del proyecto

## Comandos Útiles
- `npm run dev`: Inicia el servidor de desarrollo
- `npm run build`: Compila la aplicación para producción
- `npm start`: Inicia el servidor de producción

## Reglas de Diseño y Funcionamiento
1. **Respetar el diseño limpio y moderno actual**: No cambiar la estética minimalista y funcional que ya existe.
2. **Mantener los mismos colores (azul y oscuro)**: No modificar la paleta de colores principal.
3. **No borrar las categorías de comida que ya funcionan**: Las categorías existentes deben permanecer intactas.
4. **Cualquier cambio debe combinar con la estética actual**: Nuevas funcionalidades deben integrarse visualmente con el diseño existente.
